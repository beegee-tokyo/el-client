§ registerCallback()

void ELClientWebServer::registerCallback ( )

registerCallback()

Register callback function for incoming request

Note
WebServer doesn't send messages to MCU only if asked. Register here to the web callback. Periodic reregistration is required in case of ESP8266 reset
Example
if ((millis()-last) > 4000) {
webServer.registerCallback(); // just for esp-link reset: reregister callback at every 4s
last = millis();
}
